# Glossário

[← Voltar ao Índice](README.md)

---

## Termos Gerais

| Termo | Definição |
|-------|-----------|
| **ADM** | Administrador - usuário com permissões elevadas para gerenciar conteúdo e eventos |
| **Badge** | Distintivo visual representando conquista ou status |
| **Check-in** | Registro de presença em evento via escaneamento de QR Code |
| **Common User** | Usuário comum - membro da associação com permissões padrão |
| **Display** | Tela de TV/monitor em modo kiosk para exibição de eventos |
| **Feed** | Fluxo de posts de usuários em ordem cronológica |
| **MVP** | Minimum Viable Product - versão mínima viável do produto |
| **QR Code** | Código de barras bidimensional escaneável por câmera |
| **RSVP** | Confirmação de presença em evento |
| **Story** | Conteúdo efêmero visível por 24 horas |
| **Tooltip** | Informação contextual exibida ao interagir com elemento |

---

## Termos Técnicos

| Termo | Definição |
|-------|-----------|
| **API** | Application Programming Interface - interface de comunicação entre sistemas |
| **Cache** | Armazenamento temporário de dados para acesso rápido |
| **Endpoint** | URL específica de uma API para uma operação |
| **Fallback** | Comportamento alternativo quando o principal falha |
| **Haptic Feedback** | Vibração do dispositivo como resposta a interações |
| **Kiosk Mode** | Modo de exibição em tela cheia sem controles de navegação |
| **Lazy Loading** | Carregamento de conteúdo sob demanda |
| **Polling** | Verificação periódica de atualizações no servidor |
| **Push Notification** | Notificação enviada pelo servidor ao dispositivo |
| **Skeleton** | Placeholder visual durante carregamento de conteúdo |
| **WebSocket** | Protocolo de comunicação bidirecional em tempo real |
| **WCAG** | Web Content Accessibility Guidelines - diretrizes de acessibilidade |

---

## Termos de UX/UI

| Termo | Definição |
|-------|-----------|
| **Bottom Sheet** | Modal que surge da parte inferior da tela |
| **Breadcrumb** | Indicador de localização na hierarquia de navegação |
| **Card** | Componente visual que agrupa informações relacionadas |
| **Dropdown** | Menu que expande ao clicar |
| **Lightbox** | Overlay para visualização de imagens em tela cheia |
| **Modal** | Janela sobreposta que requer interação |
| **Onboarding** | Fluxo de introdução para novos usuários |
| **Overlay** | Camada visual sobreposta ao conteúdo principal |
| **Pull-to-Refresh** | Gesto de arrastar para baixo para atualizar conteúdo |
| **Scroll Infinito** | Carregamento contínuo de conteúdo ao rolar |
| **Shimmer** | Efeito de brilho em skeletons de carregamento |
| **Toast** | Mensagem temporária não intrusiva |
| **Toggle** | Controle de liga/desliga (switch) |

---

## Termos de Negócio

| Termo | Definição |
|-------|-----------|
| **Associação** | Organização/clube que utiliza o A-hub |
| **Benefício** | Desconto ou vantagem oferecido por parceiro |
| **Convênio** | Acordo com estabelecimento parceiro |
| **Gamificação** | Uso de elementos de jogos para engajamento |
| **Membro** | Usuário associado à organização |
| **Parceiro** | Estabelecimento que oferece benefícios aos membros |
| **Pontos** | Moeda virtual do sistema de gamificação |
| **Ranking** | Classificação de usuários por pontuação |
