# Perfil - Critérios de Aceitação

[← Voltar ao Índice](README.md)

---

## Visualização

- [ ] Usuário consegue visualizar perfil próprio
- [ ] Usuário consegue visualizar perfil de outros
- [ ] Foto de perfil é exibida corretamente
- [ ] Nome e username são exibidos
- [ ] Badges são exibidos (máx 3 visíveis)
- [ ] Posts são exibidos em grid

---

## Edição

- [ ] Usuário consegue editar próprio perfil
- [ ] Usuário consegue alterar foto
- [ ] Usuário consegue selecionar 3 badges para exibir

---

## Interações

- [ ] Usuário consegue visualizar posts em grid
- [ ] Usuário consegue abrir posts individualmente
- [ ] Usuário consegue enviar mensagem para outros usuários
- [ ] Stories são acessíveis pela foto de perfil (se houver)

---

## Badges

- [ ] Modal de badges abre corretamente
- [ ] Todos os badges são listados
- [ ] Descrição de cada badge é exibida
- [ ] Seleção de badges para exibir funciona

---

## Estados

- [ ] Loading states funcionam (skeleton)
- [ ] Estado vazio de posts é amigável
- [ ] Erros são tratados adequadamente
